Conclusão:

Para usarmos uma frase em Negrito utilizamos a tag "strong"

Para usarmos uma frase em Itálico utilizamos a tag "em"

Para usarmos uma frase em sublinhado utilizamos a tag "ins" ou "u"

Para usarmos uma frase marcada, utilizamos a tag "mark"

Para deixarmos um texto pequeno ou grande, usanmos respectivamente as tags ""small" e "big"

para deixar o texto com uma linha no meio das palavras utilizamos a tag "del"

Para elevar um número ou letra utilizamos a tag"sup"

Para deixarmos uma letra abaixo das outras utilizamos a tag "sub"
